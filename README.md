# The Fifty Project

**The Fifty Project** is a transparent, impact-driven charitable initiative that distributes ₹1 crore (10 million rupees) across 50 vetted causes — ₹2 lakh at a time, every fortnight, chosen with evidence, proven with receipts. Every rupee is tracked, documented, and shown on camera.
